Accesibilidad en Nuestro Sitio Web

Introducción

En Urban Food, estamos comprometidos con la accesibilidad web para todos los usuarios, independientemente de sus habilidades o discapacidades. Aquí explicamos las medidas que hemos tomado para garantizar una experiencia accesible para todos.

Navegación

Navegar por nuestro sitio es sencillo y accesible. Hemos implementado menús claros y etiquetas descriptivas para facilitar la navegación y la comprensión del contenido.

Compatibilidad

Nuestro sitio web ha sido diseñado para ser compatible con una amplia variedad de dispositivos y tecnologías. Hemos realizado pruebas en diferentes navegadores y pantallas para asegurarnos de que todos los usuarios puedan acceder al contenido sin problemas.

Ayudas Técnicas

Entendemos que algunos usuarios pueden requerir ayudas técnicas, como lectores de pantalla o teclados alternativos. Hemos optimizado nuestro sitio web para funcionar de manera efectiva con estas ayudas y hemos proporcionado etiquetas apropiadas y estructura semántica.

Proporcionar Feedback

Tu opinión es valiosa para nosotros. Si encuentras algún problema de accesibilidad o tienes sugerencias para mejorar la experiencia de usuarios con discapacidades, te animamos a ponerte en contacto con nuestro equipo de soporte.
